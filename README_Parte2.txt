A la hora de iniciar la simulación de la parte 2 hay un problema con las luces,
estas no renderizan de la mejor manera, para cambiar esto es necesario:

En Unity ir a:
Barra superior del editor -> Edit -> Project Settings -> Quality

En Quality cambiar Pixel Light Count a 10 (o más)

---Advertencia---
Esto es solo para mejorar la visibilidad de las luces, no se sabe si esto puede afectar
al rendimiento, si esto sucede lo mejor es regresar al valor predeterminado.
